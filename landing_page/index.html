How The RAG Pipeline Operates

1

PDF Extraction

Pdfplumber isolates text tokens natively from uploaded JD and Resume documents, preventing corrupted context streams.

2

LLM In-Context Analysis

Mistral 7B evaluates mathematical keyword densities, rescores the ATS alignment, and explicitly rewrites the missing bullet points.

3

PDF Generation

The backend routes the raw JSON response directly through ReportLab, actively mapping data to beautiful formatted ATS templates.
